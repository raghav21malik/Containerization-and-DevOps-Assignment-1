🚀 How to Access the Files

  1. Open the Repository: Click "GitHub Repository" above — explore all source code, Dockerfiles, and configuration files.
  2. Read the Documentation: Click "Project Documentation" — full README with setup instructions and detailed explanations.
  3. View Implementation Details: Check the implementation section for Docker configs, compose files, and networking setup.
  4. Run It Locally: Clone the repo and run docker compose up --build — the application starts at localhost:3000.
